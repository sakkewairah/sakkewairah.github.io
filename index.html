What is coding

Coding, sometimes called computer programming, is how we communicate with computers. Code tells a computer what actions to take, and writing code is like creating a set of instructions. By learning to write code, you can tell computers what to do or how to behave in a much faster way. You can use this skill to make websites and apps, process data, and do lots of other cool things.

Why learn how to code?

Computer and information technology jobs are growing incredibly fast in the United States and across the globe. You don't need a Computer Science degree.


I am learning how to code because i think it's an important skill. Everybody Should learn how to code.

Make cool stuff

Coding makes it possible to make websites or even apps. Learning to code makes it possible to imagine the things you want to make and actually build them.

Coding is difficult but rewarding

I know coding might sometimes be annoying and tiring but once you get the project done it feels amazing. Coding also opens up a whole new community and network for you. In fact, we've heard the term "addictive" regularly.
